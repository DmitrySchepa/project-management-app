Перед началом работы необходимо зарегестрировать пользователя. При нажатии на кнопку регистрации, появится форма регистрации. После успешной регистрации пользователь будет перенаправлен на главную страницу приложения.

Для создания новой доски необходимо нажать кнопку "Новая доска", после этого пользователю будет предложено ввести название доски и её описание. После успешного создания, доска появится на странице.

При нажатии на доску пользовательно будет перенаправлен на текущую(выбранную) доску. При нажатии на "Добавить колонку" пользователю будет предложено поля для заполнения названия колонки.

Pomodoro

Программа для тайм-менеджмента по технике Pomodoro, когда работа выполняется в течение заданного промежутка времени (по умолчанию 25 минут), после которого следует отдохнуть. Использование программы: На главном экране отображается список помидорок (ранее учтённых промежутков времени). Чтобы стартовать новый помидор, нужно нажать "New Pomodoro" и задать название, описание и желаемый промежуток времени. После нажатия на "Start" включится таймер. Если нажать кнопку "Stop" не дожидаясь окончания промежутка времени, текущий помидор считается проваленным (Failed, окрашен оранжевым). У каждого помидора в списке есть меню (три точки), из которого можно удалить помидор, отредактировать название и описание, а также поменять статус на успешный и проваленный по желанию. Для обновления списка, нажать "Update list". Данные хранятся в local storage.
